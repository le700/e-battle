## Show HN 标题

```
Show HN: 我做了个让微信好友"克隆人"互骂的开源项目 (FriendBattle)
```

### Show HN 正文
```
大家好！

我开源了一个有趣的项目：FriendBattle

核心功能：
- 从微信/Telegram 聊天记录克隆好友的语言风格
- 用 AI 生成好友的"克隆人"
- 让两个克隆人针对任何话题进行 Battle

使用场景：
- "甜豆花好吃还是咸豆花好吃？"
- "周杰伦和林俊杰谁的歌更好听？"
- "《原神》和《王者荣耀》谁更火？"

技术栈：
- Python 3.12
- Flask
- PyWxDump（微信数据处理）
- Transformers

非常欢迎大家试用和提建议！

GitHub: github.com/le700/FriendBattle
```
